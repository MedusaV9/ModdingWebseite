# ModdingWebseite

Single-page marketing site for **BAPBAP Modding** — a community modding project
for the Steam game BAPBAP. Built with Vite, React, TypeScript, and Tailwind CSS v4.

## Quickstart

```bash
npm install
npm run dev
```

Other scripts:

```bash
npm run lint    # oxlint
npm run build   # tsc + vite build
npm run preview # preview the production build
```
